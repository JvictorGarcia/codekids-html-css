Estrutura de uma Página

Monte uma página completa com header, main e footer.

ExercícioHTML
0% Completo
☆☆☆☆☆

🎯 Objetivo

Criar uma página HTML completa com cabeçalho, conteúdo principal e rodapé.

💡 Instruções

  1. Estruture com <header>, <main> e <footer>.
  2. Adicione um <h1> e 2 parágrafos no main.
  3. Coloque informações de contato no rodapé.

🧩 Código Inicial

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Minha Página Completa</title>
</head>
<body>
  <header><h1>Bem-vindos!</h1></header>
  <main>
    <p>Este é o conteúdo principal.</p>
    <p>Mais informações aqui.</p>
  </main>
  <footer><small>© 2025 - Meu Site</small></footer>
</body>
</html>
