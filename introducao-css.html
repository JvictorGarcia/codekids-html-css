Introdução ao CSS

Como estilizar páginas com folhas de estilo.

CSSVídeo
0% Completo
☆☆☆☆☆

🎯 Objetivo

Entender seletores, propriedades e como ligar o CSS ao HTML.

💡 Instruções

  1. Crie um arquivo styles.css.
  2. Ligue no <head> com <link rel=stylesheet>.
  3. Estilize h1 e p com cores e fonte.

🧩 Código Inicial

<link rel="stylesheet" href="styles.css">
h1 { color: blue; }
p  { font-family: Arial; }
