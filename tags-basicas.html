🎯 Objetivo

Aprender títulos, parágrafos, links e imagens.

💡 Instruções

  1. Use <h1> a <h3> para títulos.
  2. Crie parágrafos com <p>.
  3. Insira links com <a href> e imagens com <img alt>.

🧩 Código Inicial

<h1>Meu Título</h1>
<p>Um parágrafo com <a href="#">um link</a>.</p>
<img src="foto.jpg" alt="Minha foto">
